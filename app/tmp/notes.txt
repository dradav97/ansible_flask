in this folder there is file env.j2 
.j2 is extension to ninja documents

is a template to create the file with secrets

the security variables init with <<SEC_>>
the general variables init with <<VAR_>> and are in "envs" folder in the file <<common.yml>>
the variables obtained from terraform init with <<TERRAFORM_VAR_>> (this variables is generated by terraform inform)


 
Jinja is a fast, expressive, extensible templating engine. Special placeholders in the template allow 
writing code similar to Python syntax. Then the template is passed data to render the final document.
